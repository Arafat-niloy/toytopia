## ðŸŽ¯ **Project Purpose**
The main goal of this project is to create a responsive and interactive toy marketplace using **React** and **Firebase Authentication**.  
Users can:
- Browse toys from different sellers.
- Log in or register securely.
- Reset their passwords.
- Leave reviews or feedback for products.

This project demonstrates key concepts of React such as components, props, hooks, and Firebase integration for authentication.

## âœ¨ **Key Features**

### Authentication
- Login and Registration with Email/Password.
- Google Sign-In using Firebase.
- Forgot Password functionality that redirects users to Gmail after reset.

### Toy Management
- View toy cards with images, descriptions, and prices.
- Detailed toy information pages.
- Add new toys (for sellers).
- Dynamic routing for toy details.

## ðŸš€ Technologies & NPM Packages Used

This project was bootstrapped with **Vite**.

### Core Frameworks & Libraries:

* **React.js:** The core library for building the user interface.
* **React Router DOM:** For client-side routing and navigation (`v6+`).
* **Firebase:** Used for all authentication services (User/Pass, Google Sign-In, Password Reset, Profile Updates).

### Styling:

* **Tailwind CSS:** A utility-first CSS framework for rapid UI development.
* **DaisyUI:** A component library for Tailwind CSS to quickly build components like Navbars, Footers, Buttons, and Cards.

### Utility Packages:

* `react-helmet-async`: Manages dynamic page `<head>` titles for better SEO and UX.
* `react-hot-toast`: Provides simple and clean toast notifications.
* `aos`: Adds animations on scroll.
* `react-icons`: Provides a library of icons (e.g., `FaEye`, `FaEyeSlash`) for the UI.
